saraswati is a fast, native Markdown reader and editor for your Mac. Open any folder as a workspace, write in rich text or raw Markdown, and every file stays a plain .md on disk. Free.

02 — four you'll tell a friend about

  1. 01

    slides from plain markdown.

    Add marp: true and a note becomes a deck — preview beside the source, present fullscreen, export to PDF.

  2. 02

    typeset, not just rendered.

    An editorial serif, accent-barred headings, code on dark slabs. Your pages are set like print, not dumped like a preview.

  3. 03

    frontmatter without the yaml.

    Metadata becomes chips you edit. Fields you don't touch round-trip byte-for-byte, so diffs stay clean.

  4. 04

    writes both ways.

    Rich text or raw Markdown — switch whenever; every file stays a plain .md.

And the rest of the kit: workspaces from any folder, full-text search, a live outline, inline Mermaid, and side-by-side diffs.
