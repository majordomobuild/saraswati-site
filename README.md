# saraswati — concept rebrand site

A self-contained static landing page that rebrands korq as **saraswati** — a soft "Lavender"
identity (pale lavender ground `#F5F3FC`, deep plum ink, one refined violet accent `#5B48D6`, a
**Newsreader** serif wordmark/headline set, and the reference violet lotus reproduced faithfully as
the mark) over the same product story and editorial page architecture as the korq site. saraswati
ships no binary of its own, so every Download / releases CTA points at the current korq build
(`https://github.com/majordomobuild/releases/releases/latest`) and the footer says so plainly.

Open `index.html` in a browser, or serve the folder with any static server
(`python3 -m http.server`) — the only external dependency is Google Fonts.
